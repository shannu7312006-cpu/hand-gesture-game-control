# Hand Gesture Controlled System

Project Description:

This project is a real-time hand gesture recognition system that allows users to control applications without using a keyboard or controller. It uses a webcam along with OpenCV and MediaPipe to detect hand gestures and convert them into keyboard actions like acceleration and braking.

# Features
1. Real-time hand tracking
2. Finger counting using landmarks
3. Gesture-based control (GAS / BRAKE / NEUTRAL)
4. Touchless interaction
5. Works with standard webcam

# Technologies Used
Python
OpenCV
MediaPipe
CV2 (for video processing)
DirectKeys (for keyboard simulation)

# How to run 
* pip install opencv-python mediapipe
* python your_file_name.py
